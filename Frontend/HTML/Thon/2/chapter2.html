HyperText Markup Language

HTML is the standard language used to create web pages. It's not a programming language; it's a markup language. This means it uses tags to "mark up" or structure the content, telling the browser how to display text, images, and other elements.

  • HyperText: Refers to the "links" that connect web pages to one another, allowing the user to navigate.
  • Markup: Refers to the tags used to define the structure and meaning of the content.

Fundamental Content Tags

Headings (<h1> to <h6>)

Headings are used to define a hierarchy of importance for titles and subtitles on a page. Search engines use them to understand the structure of your content.

This is an H1 heading (Most Important)

This is an H2 heading

This is an H3 heading

This is an H4 heading

This is an H5 heading
This is an H6 heading (Least Important)
Best Practice: Use only one <h1> per page for the main title. Don't skip heading levels (e.g., don't go from an <h2> to an <h4>).

Paragraphs (<p>)

The <p> tag is used to group text into paragraphs. The browser automatically adds some space before and after each paragraph.

This is the first paragraph. Lorem ipsum dolor sit amet, consectetur adipiscing elit.

This is the second paragraph. It is separate from the first one.

Breaks (<br>) and Horizontal Rules (<hr>)

These are "self-closing" tags.

  • The <br> tag creates a line break.
    As you can see, this text is on a new line.
  • The <hr> tag creates a thematic break or a horizontal rule, often used to separate sections of content.

Lists: Organizing Information

Unordered List (<ul>)

Used for lists where the order of items does not matter. It uses bullet points by default.

  • Milk
  • Tea Leaves
  • Sugar

Ordered List (<ol>)

Used for lists where the order is important, like a recipe or a ranking. It uses numbers by default.

  1. Boil water.
  2. Add tea leaves.
  3. Add milk and sugar.

Links and Images

Anchor Tag (<a>)

The anchor tag is used to create hyperlinks. The href attribute specifies the destination URL.

Visit the main page of our project: Go to Home.

To open a link in a new tab, use the target="_blank" attribute: Search on Google.

Image Tag (<img>)

The image tag is used to embed an image. It is a self-closing tag and requires the src (source) and alt (alternative text) attributes.
