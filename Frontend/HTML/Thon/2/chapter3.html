HTML Tables

Tables are used to display data in a structured, grid-like format of rows and columns.

Basic Table Structure

  • <table>: The container for the entire table.
  • <tr> (Table Row): Defines a row in the table.
  • <th> (Table Header): Defines a header cell. Text is bold and centered by default.
  • <td> (Table Data): Defines a standard data cell.

Advanced Structure & Spanning

For more complex and semantically correct tables, we use additional tags and attributes:

  • <thead>, <tbody>, <tfoot>: These tags group the header, body, and footer content of a table, respectively. They improve accessibility and code organization.
  • colspan: An attribute to make a cell span across multiple columns.
  • rowspan: An attribute to make a cell span across multiple rows.
