What are Forms?

HTML forms are one of the most important parts of the web. They are used to collect user input, such as login details, registration information, search queries, and more. A form is a container for different types of input elements.

Key Form Concepts

  • <form>: The main container for the form. The action attribute tells the browser where to send the data when the form is submitted (usually a backend URL).
  • <input>: The most versatile form element. Its behavior changes based on its type attribute.
  • <label>: Provides a caption for an input element. Using the for attribute, which matches the id of an input, links them together. This improves accessibility and allows users to click the label to focus on the input.
  • The name Attribute: This is CRITICAL. The name attribute is the key that gets sent to the backend with the user's input as the value. Without a name, the data from that input field is not submitted.
Hinglish Mnemonic: "Bina naam ke, data bekaar hai." (Without a name, the data is useless). The backend identifies data by its name.
