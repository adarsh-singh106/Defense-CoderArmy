Chapter 1: The Beginning - How the Internet Works

What is the Internet?

The internet is essentially a global network of computers that communicate with each other. Think of it like a massive postal service. To send a package (data), you need an address. In the digital world, this address is called an IP Address.

IP Address (Internet Protocol)

An IP address is a unique numerical label assigned to every device connected to a computer network. It's like the home address for your computer or phone. It allows devices to find and communicate with each other.

IPv4 vs. IPv6

  • IPv4: An example is 12.3.45.6. It uses a 32-bit format, which allows for about 4.3 billion unique addresses. We've nearly run out of these!
  • IPv6: A newer, 128-bit version that allows for a virtually limitless number of addresses. This is the future of the internet.

DNS (Domain Name System)

Remembering IP addresses like 8.8.8.8 is hard for humans. We prefer names like google.com. The DNS is the phonebook of the internet. When you type a website name (domain name) into your browser, the DNS server looks up the corresponding IP address so your browser can load the website.

How DNS Works (Simplified):

  1. You type coderarmy.in in your browser.
  2. Your browser asks a DNS server, "What is the IP address for coderarmy.in?"
  3. The DNS server replies with the IP address.
  4. Your browser uses that IP address to request the website's content.

MAC Address (Media Access Control)

If an IP address is like your home's street address (which can change if you move), the MAC address is like your house's unique serial number. It's a permanent, physical address assigned to the network hardware of your device.

Why do we need both? Imagine your Wi-Fi router (ISP) assigns your phone an IP address. If you disconnect, that IP might be given to another device. When data comes back, the network needs to be sure it goes to the correct physical device. The MAC address ensures the data reaches your specific phone, not your dad's phone which might have temporarily gotten the same IP. Hinglish Mnemonic: IP address 'Dynamic' hai, MAC address 'Permanent' hai. Simple!

Port Numbers

A port number is used to identify a specific application or service on your device. Your computer has one IP address, but you might be browsing the web, checking email, and listening to Spotify all at once. How does the data know where to go?

Think of your IP address as an apartment building's address. The port number is the apartment number. When data arrives, the port number directs it to the correct application (e.g., Port 443 for secure websites, Port 80 for standard websites).
