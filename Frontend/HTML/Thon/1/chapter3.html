Chapter 3: Organizing Data with Nested Lists & Tables

Nested Lists

A nested list is simply a list inside another list. This is useful for creating outlines or hierarchical structures.

Example: Countries and Cities

  • India
    • Delhi
    • Mumbai
    • Goa
  • USA
    1. New York
    2. California

HTML Tables

Tables are used to display data in a structured, grid-like format of rows and columns.

Basic Table Structure:

  • <table>: The container for the entire table. The border="1" attribute adds a simple border.
  • <tr>: Table Row. Defines a row of cells.
  • <th>: Table Header. Defines a header cell. Text is typically bold and centered.
  • <td>: Table Data. Defines a standard data cell.

Example: Student Information

Name Role Project
Rohit Frontend Developer Notes Website
Mohit Backend Developer API Service
Anjali Full-Stack Developer E-commerce Platform
This is a list of top students.

Advanced Tables: Rowspan and Colspan

These attributes allow you to merge cells across multiple rows or columns.

  • colspan: Merges a cell with cells to its right (horizontally).
  • rowspan: Merges a cell with cells below it (vertically).
