Chapter 2: What is HTML (HyperText Markup Language)?

HTML is the Skeleton

HTML is not a programming language. It's a markup language. Its job is to give structure and meaning to the content of a webpage. Think of it as the skeleton of a human body. It provides the basic structure, but it's not styled (that's CSS) and it doesn't have functionality (that's JavaScript).

Basic Tags

Headings (h1 to h6)

Headings are used to define the importance of a title. <h1> is the most important (usually the main title of a page), and <h6> is the least important.

This is an h1 heading.

This is an h2 subheading.

This is an h3 subheading.

Paragraphs (<p>)

The paragraph tag is used for blocks of text, like the one you are reading right now.

This is another paragraph. Each 'p' tag creates a new block of text with some space around it.

Line Breaks (<br>) and Horizontal Rules (<hr>)

A <br> tag creates a line break.
As you can see, this text is on a new line.
It's a self-closing tag.

An <hr> tag creates a thematic break or a horizontal line, like the ones used to separate sections on this page. It is also self-closing.

Lists

Unordered List (<ul>)

Used for lists where the order doesn't matter. It uses bullet points.

  • Milk
  • Bread
  • Sugar

Ordered List (<ol>)

Used for lists where the order is important, like steps in a recipe. It uses numbers.

  1. Boil water.
  2. Add tea leaves.
  3. Add milk and sugar.

Anchor Tag (<a>) - Creating Links

The anchor tag is used to create hyperlinks to other pages or websites. The href attribute specifies the destination URL.

Click here to visit Google
Click here to go back to Chapter 1

Image Tag (<img>)

The image tag is used to embed an image on a page. It is self-closing. The src attribute specifies the path to the image, and the alt attribute provides alternative text if the image cannot be displayed.

Note: Since we can't include actual images in this HTML-only project without a file structure, here is how the code would look:
