Chapter 4: HTML Boilerplate & Key Fundamentals

File Path System

File paths tell the browser where to find files (like images, stylesheets, or other HTML pages). There are two types:

  • Absolute Path: A full URL to a file hosted on the internet. It starts with https://. This is used for external resources.
    Example: <img src="https://www.google.com/images/logo.png">
  • Relative Path: A path to a file relative to the current page's location. This is used for internal files within your project.
    • image.jpg: The file is in the same folder.
    • images/image.jpg: The file is in an 'images' folder at the same level.
    • ../image.jpg: The file is in the parent folder (one level up).

The HTML Boilerplate Explained

The boilerplate is the standard, essential code required for every HTML page to work correctly across different browsers.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    ... your content here ...
</body>
</html>
  • <!DOCTYPE html>: Declares that this document is an HTML5 document.
  • <html lang="en">: The root element of the page. lang="en" specifies the page language as English.
  • <head>: Contains meta-information about the document (data about data). This is not displayed on the page.
  • <meta charset="UTF-8">: Specifies the character encoding for the document. UTF-8 includes almost all characters and symbols in the world.
  • <meta name="viewport" ...>: Makes the website responsive, ensuring it looks good on all devices (desktops, tablets, and phones).
  • <title>: The title of the page, which appears in the browser tab.
  • <body>: Contains the visible content of the webpage. Everything you see is inside the body tag.

Div, Class, and ID

Div (<div>) - The Container

A <div> is a generic container element. It doesn't do anything on its own, but it's used to group other elements together. This is extremely useful for applying styles (with CSS) or manipulating a group of elements at once (with JavaScript). Hinglish Mnemonic: Div ek 'dabba' hai jisme aap kuch bhi daal sakte ho.

Class and ID - Selectors

Class and ID are attributes you can add to any HTML element to identify them. This is crucial for CSS and JavaScript.

  • Class: Used to group multiple elements. The same class name can be used on many elements. Think of it as labeling all students in the 'Frontend' batch.
    Example: <p class="highlight">...</p> and <h2 class="highlight">...</h2>
  • ID: Used to identify a single, unique element. An ID name must be unique within the entire page. Think of it as a student's unique roll number.
    Example: <div id="main-header">...</div>
