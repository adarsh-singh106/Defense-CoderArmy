Media Tags: Audio and Video

HTML5 introduced dedicated tags for embedding audio and video files directly into your webpages without needing plugins.

The <audio> Tag

Used to embed sound content. The controls attribute adds play, pause, and volume controls.

<audio controls>
    <source src="path/to/song.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

Using multiple <source> tags provides fallback options for different browsers.

The <video> Tag

Used to embed a video player. Like audio, controls adds player controls. You can also set width and height.

<video width="400" controls>
    <source src="path/to/video.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

Embedding YouTube Videos with <iframe>

An <iframe> (Inline Frame) is used to embed another HTML document within the current one. This is the standard way to embed content from external sites like YouTube or Google Maps.

How to Embed a YouTube Video:

  1. Go to the YouTube video you want to embed.
  2. Click the "Share" button below the video.
  3. Click on the "Embed" option.
  4. Copy the provided iframe code and paste it into your HTML.

Building a Multi-Page Website

A multi-page website is just a collection of HTML files linked together. This project is a perfect example!

Structure:

  • index.html: The homepage.
  • about.html: The about us page.
  • contact.html: The contact page.

Linking Pages:

You use the anchor <a> tag with a relative path to link your pages. For example, on your homepage (index.html), you would link to the about page like this:

<a href="about.html">About Us</a>

This creates the navigation that allows users to move between the different sections of your site.

Deploying a Static Website

Deployment is the process of making your website live on the internet so anyone can access it. For simple HTML websites (static sites), you can use free services.

Popular Free Deployment Platforms:

  • Netlify: Famous for its simple drag-and-drop deployment. You can drag your project folder directly onto their site, and it will be live in seconds.
  • GitHub Pages: If your code is hosted on GitHub, you can deploy it as a website with a few clicks.
  • Vercel: Another excellent platform with a generous free tier, very popular in the developer community.

The process is usually as simple as signing up, linking your code repository or uploading your files, and the platform handles the rest!
